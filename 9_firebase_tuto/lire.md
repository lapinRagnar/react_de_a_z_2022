- créer un compte sur firebase.fr
- créer un projet : firebase-tutorial
- va dans project settings:

- dans Cloud Firestore:
    * va dans règles
    * remplacer : allow read, write: if false; en allow read, write: if true;
    * appuyer sur Publish

- dans data
